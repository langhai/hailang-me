Links

You can easily create a hyper link by adding a '_' after a word.

This is a link_

In order for a hyperlink to work, you'll have to define it's destination as well. The link destination can be defined anywhere in the document, putting all hyperlink definitions at the end of your document seems to be a nice idea.

Click here_ for google

.. _here: http://google.com

Here's how it looks like, now 'here' is a hyperlink to google.com Click here for google

Code

You can include code snippets in your document by using code or code-block directives.

.. code-block:: identifier
<blank line>
<indent>#include <stdio.h>
<indent>
<indent>void take_out_garbage() {
<indent>    printf("Yes, ma'am!\n");
<indent>    sleep(5000);
<indent>}
<blank line>

And this will look like

#include <stdio.h>

void take_out_garbage() {
    printf("Yes, ma'am!\n");
    sleep(5000);
}
